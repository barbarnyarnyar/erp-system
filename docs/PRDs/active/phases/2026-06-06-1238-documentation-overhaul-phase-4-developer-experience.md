## Verification

### Automated

```bash
# Check for broken markdown links
grep -rn '](..*\.md)' documentation/ | grep -v 'README.md' | \
  while IFS= read -r line; do
    file=$(echo "$line" | sed -n 's/.*](\([^)]*\.md\))/\1/p')
    dir=$(echo "$line" | cut -d: -f1 | xargs dirname)
    [ -n "$file" ] && [ ! -f "$dir/$file" ] && [ ! -f "$file" ] && echo "BROKEN: $line"
  done
```
